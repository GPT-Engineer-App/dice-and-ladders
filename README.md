# dice-and-ladders

A snakes and ladders game with HTML5 and CSS3, featuring a 100-square board. The game includes a throwing dice animation at each turn, options for player vs. PC or multiplayer modes, and a background of a Japanese mountain village view.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/dice-and-ladders.git

```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
